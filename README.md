# Titanic Survival Prediction

## Description
Predict Titanic passenger survival using machine learning.

## Dataset
- **Survived**: 1 = Survived, 0 = Did not survive
- **Pclass**: Passenger class
- **Age**: Age of passenger
- **Sex**: Gender
- **Fare**: Fare paid
- **Embarked**: Port of embarkation

## Usage
1. Load the data.
2. Train the model.
3. Predict survival.
